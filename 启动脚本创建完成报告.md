## 🔍 故障排除指南

### 常见问题及解决方案

#### 1. 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr :3000

# 终止占用进程
taskkill /f /pid <进程ID>
```

#### 2. 数据库连接失败
```bash
# 检查MySQL服务
wsl -d Ubuntu-22.04 --exec sudo systemctl status mysql

# 启动MySQL服务
wsl -d Ubuntu-22.04 --exec sudo systemctl start mysql
```

#### 3. 依赖包缺失
```bash
# 重新安装依赖
cd backend && npm install
cd ../frontend && npm install
cd ../admin && npm install
```

#### 4. WSL服务未启动
```bash
# 重启WSL
wsl --shutdown
wsl -d Ubuntu-22.04
```
